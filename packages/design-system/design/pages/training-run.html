Вопрос 4 · что вернёт запрос?
-- orders: 5 строк, 2 без customer_id
SELECT COUNT(*) FROM orders o
LEFT JOIN customers c ON o.customer_id = c.id
WHERE c.country = 'RU';
// объяснение

Условие WHERE c.country = 'RU' отсеивает строки, где c пустой (NULL после LEFT JOIN), превращая внешнее соединение во внутреннее. Чтобы сохранить заказы без клиента, перенесите условие в ON.
